# Release v0.1.9

**Date:** 2026-06-02
**Type:** Release

## Summary

Cut v0.1.9 to ship the paper-knowledge workflow + the two CI/release fixes on a
tag whose commit includes the corrected workflows (a v0.1.8 re-run could not
pick up the fixes — release workflows build from the tag's commit).

## What's in this release

- **feat:** in-app "Build Knowledge & Write Paper" workflow + paper pattern/gap
  detection (`2026-06-01_19`).
- **fix(ci):** `cargo check` cleared `bundle.resources` (onedir glob) so the CI
  Rust job passes (`2026-06-02_01`).
- **fix(release-mac):** pre-sign every onedir Mach-O with Developer ID +
  hardened runtime + secure timestamp before the bundle, so notarization no
  longer returns `Invalid` (`2026-06-02_02`).

## Changes

- Bumped version `0.1.8` → `0.1.9` in `tauri.conf.json`, `Cargo.toml`,
  `Cargo.lock`, `package.json`, `package-lock.json`.
- Tagged `v0.1.9` → triggers `release-mac.yml` / `release-windows.yml` /
  `release-linux.yml` (all trigger on `push: tags: v*`).
